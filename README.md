# word-ranking-api

## バージョン
- Python: 3.9.16
- Flask: 2.2.3
- SudachiPy: 0.6.7

## 環境構築
```
$ git clone git@github.com:Shuma-Yamamoto/word-ranking-api.git
$ cd word-ranking-api
$ python3 -m venv venv
$ source venv/bin/activate
```

## ライブラリ
```
$ pip install --upgrade pip
$ pip install flask
$ pip install sudachipy
$ pip install sudachidict_core
```
